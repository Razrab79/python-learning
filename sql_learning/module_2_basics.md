# Модуль 2: Базовые запросы

Добил второй модуль на тренажере. Синтаксис пока не сложный, постепенно привыкаю к логике базы данных.

Что разобрал:
* Базовый вытаск данных: `SELECT`, исключение дублей `DISTINCT`.
* Фильтрация: `WHERE` и всякие операторы типа `IN`, `BETWEEN`, `LIKE` (поиск по тексту).
* Сортировка: `ORDER BY`.
* Агрегатные функции: `COUNT`, `SUM`, `AVG` и т.д.
* Группировка: `GROUP BY` и `HAVING`. 

**Главный затык, который сегодня понял:** 
Если используешь агрегатную функцию (например, считаешь среднее через `AVG`), то все остальные обычные колонки в `SELECT` обязательно нужно засовывать в `GROUP BY`. Иначе база не поймет, как схлопывать строки.

Просто шпаргалка, чтобы не забыть структуру:
```sql
SELECT room_id, AVG(price) AS avg_price, COUNT(*) AS count
FROM Reservations
GROUP BY room_id
HAVING COUNT(*) > 2
ORDER BY count DESC;
